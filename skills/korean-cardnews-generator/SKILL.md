# 한국어 카드뉴스 생성기 (Korean Card News Generator)

## 목적

마케터·1인 사업가·콘텐츠 크리에이터가 주제 하나만 던지면, 바로 디자인 툴에 옮길 수 있는 **인스타그램/카카오채널 카드뉴스 구성안**을 완성합니다.

## 사용 시점

- 인스타그램 정보성 카드뉴스를 빠르게 기획할 때
- 긴 블로그 글/보고서를 SNS 카드뉴스로 재가공할 때
- 카카오채널·네이버 포스트 콘텐츠를 만들 때

## 출력 형식

총 **7~9장** 구성으로, 각 장마다 다음을 제공합니다:

```
[표지]
- 헤드라인: 스크롤을 멈추게 하는 한 줄 (15자 이내, 숫자/호기심 활용)
- 서브: 보조 설명 한 줄
- 시각 가이드: 배경색·이미지 분위기·폰트 톤 제안

[본문 1~6]
- 소제목: 각 장의 핵심 (10자 이내)
- 본문: 2~3문장, 모바일에서 읽기 쉽게
- 시각 가이드: 아이콘/일러스트/도표 제안

[마무리]
- 핵심 요약 + CTA (저장/팔로우/링크 클릭 유도)
- 해시태그 8~12개 (한국어 + 영어 혼합)
```

## 작성 원칙

1. **3초 후킹** — 표지에서 멈추지 않으면 끝. 숫자·반전·공감 우선.
2. **한 장 한 메시지** — 카드 한 장에 개념 하나만.
3. **모바일 가독성** — 본문은 짧게. 긴 문장 금지.
4. **저장 욕구 자극** — "나중에 다시 볼 가치"가 있게 구성.
5. **한국 SNS 정서** — 과한 영어·번역체 금지. 자연스러운 한국어 트렌드 톤.

## 예시 (입력: "직장인 번아웃 극복법")

**[표지]**
- 헤드라인: 번아웃, 의지 문제가 아닙니다
- 서브: 매일 지치는 당신을 위한 5가지 회복법
- 시각: 차분한 네이비 배경, 지친 표정 일러스트, 부드러운 산세리프

**[본문 1] 신호 알아채기**
- 일요일 밤마다 가슴이 답답하다면 이미 시작된 겁니다.
- 시각: 체크리스트 아이콘 3개

**[본문 2] 완벽주의 내려놓기**
- "80점이면 제출"을 규칙으로 만드세요. 100점은 함정입니다.
- 시각: 80% 게이지 그래픽

*(본문 3~5 계속)*

**[마무리]**
- 번아웃은 멈추라는 몸의 신호입니다. 오늘 하나만 실천해보세요.
- CTA: 저장하고 지친 날 다시 꺼내보세요 📌
- #직장인 #번아웃 #자기계발 #워라밸 #멘탈관리 #burnout #selfcare
