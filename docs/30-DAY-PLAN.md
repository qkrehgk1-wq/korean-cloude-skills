## 🎯 핵심 정보

- **타깃 고객:** 하루 2~5시간 반복 문서 작업(보고서·회의록·이메일)을 처리하는 한국 직장인·스타트업 팀원 (검증 채널: 직장인 오픈카톡방, 커리어리, 링크드인 KR, 퇴근길 앱/토스피드 커뮤니티)
- **핵심 문제:** Claude를 써도 한국 직장 문화에 맞는 말투·양식이 없어서 매번 프롬프트를 처음부터 짜야 한다
- **쐐기 전략:** GitHub 무료 공개 스킬 5종으로 SEO·스타 트래픽 선점 → 월 $9 Flat 구독으로 프리미엄 스킬 15종+업데이트 판매
- **30일 성공 지표:** 30일 후 Gumroad 유료 구독자 6명 이상 → MRR $54 이상 (월 $9 × 6명)
- **킬 기준:** 4주차 종료 시점에 (1) GitHub Star 50개 미만 AND (2) 유료 구독자 0명 AND (3) 이메일 대기자 10명 미만 이면 프로젝트 중단. 투입 총비용 $12 이하이므로 Kelly 원칙상 손실 허용 한도 내이나, 시간 기회비용(주 10시간 × 4주 = 40시간) 회수가 불가능하다고 판단되면 즉시 피벗 또는 종료.
- **예상 월 비용:** $12
- **기술 스택:** GitHub (오픈소스 무료 스킬 배포), Gumroad (구독 결제, 수수료 10% 이하), Notion (랜딩페이지 + 문서), Vercel (정적 랜딩 사이트, 무료 티어), Claude API (스킬 테스트 환경), GitHub Actions (README 자동 업데이트), Cloudflare (DNS + 무료 CDN)
